Flags TCP/IP

  • RST: Reinicia la conexión si se recibe algo inesperado.
  • PSH: Indica que los datos deben procesarse inmediatamente.
  • URG: Indica datos urgentes prioritarios.
  • SYN: Inicia una conexión.
  • ACK: Confirma recepción de datos.
  • FIN: Finaliza una conexión.

Comprobación Manual de Servicios

  • Netcat: 
    nc -vn <ip> <puerto>
nc -vnz -w 1 <ip> 20-25
  • Telnet: telnet 192.168.0.1 25
  • HTTP: curl http://host o wget http://host
  • FTP: ftp 192.168.0.1
  • SSH: ssh user@host
  • SCP: scp archivo user@host:/ruta

Nmap NSE Scripts

Vulners

  • Busca vulnerabilidades CVE con CVSS
  • nmap -sV --script vulners <target>

VulScan

  • Requiere instalación: GitHub Vulscan
  • nmap -sV --script=vulscan/vulscan.nse <target>

Herramientas de Descubrimiento ARP

arp-scan

  • sudo arp-scan -I eth0 -l: escanea toda la red local
  • man arp-scan: abre el manual completo

netdiscover

  • sudo netdiscover -i eth0 -r 192.168.1.0/24: escanea red definida

Netcat en Metasploitable

  • nc -lp 8000: escucha por el puerto 8000
  • nc 10.0.2.21 8000: se conecta desde otro host
  • Se puede enviar texto como: echo "3 9000" | nc <host> <puerto>
