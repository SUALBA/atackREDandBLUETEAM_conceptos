Guía Compacta: Recuperar Conectividad de Red

1. Listar interfaces

ip link show

2. Activar interfaz

ip link set dev <iface> up

3. Solicitar DHCP

dhclient <iface>

4. Ver IP

ip addr show <iface>

5. Ver rutas

ip route show

6. Añadir ruta

ip route add default via <gateway> dev <iface>

7. Probar conectividad

ping 127.0.0.1 && ping <gateway> && ping 8.8.8.8 && ping google.com

8. Configurar red VM

Modo Bridged / NAT / Host-only y DHCP habilitado
