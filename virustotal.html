7. API & Ejemplos Prácticos

7.1. Enviar Lotes desde Bash

#!/bin/bash
APIKEY="TU_API_KEY"
for file in /muestras/*.exe; do
  echo "Escaneando $file..."
  curl -s -X POST "https://www.virustotal.com/api/v3/files" \
    -H "x-apikey: $APIKEY" \
    -F "file=@$file" \
    | jq '.data.id'
done

7.2. Script en Python

import requests
APIKEY = 'TU_API_KEY'
HEADERS = {'x-apikey': APIKEY}
url = 'https://www.virustotal.com/api/v3/files'
for path in ['/muestras/shell1.exe','/muestras/shell2.php']:
    with open(path,'rb') as f:
        files = {'file': (path, f)}
        resp = requests.post(url, headers=HEADERS, files=files)
        print(path, resp.json()['data']['attributes']['last_analysis_stats'])

7.3. Consultas Periódicas con cron

Ejecuta el script Bash cada hora:

# crontab -e
0 * * * * /usr/local/bin/vt_batch_scan.sh >> /var/log/vt_scan.log 2>&1

7.4. Webhooks & Alertas SIEM

Configura un endpoint para recibir notificaciones de nuevos análisis:

# server.py (Flask)
from flask import Flask, request
app = Flask(__name__)
@app.route('/vt-webhook', methods=['POST'])
def vt_webhook():
    data = request.json
    # Extrae hash y detecciones
    file_id = data['data']['id']
    stats = data['data']['attributes']['last_analysis_stats']
    # Envíalo a tu SIEM (ej. syslog)
    print(f"Nuevo informe {file_id} -> {stats}")
    return '', 204

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)

En la consola de VirusTotal (Dashboard → Webhooks), añade tu URL https://tu.dominio/vt-webhook y selecciona eventos de scan.report.
