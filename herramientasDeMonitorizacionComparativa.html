📚 Glosario de Conceptos Clave

NRPE: Plugin de Nagios que permite ejecutar comandos en máquinas remotas para monitorización avanzada.
SNMP: Protocolo de red usado para recopilar información y configurar dispositivos de red como routers, switches, etc.
NagVis: Herramienta visual que permite representar gráficamente el estado de los hosts monitorizados por Nagios.
API REST: Interfaz que permite integrar y controlar sistemas usando peticiones HTTP. Muy común para automatizar monitorización.
Webhooks: Métodos para recibir notificaciones automáticas entre sistemas cuando ocurre un evento específico.
TSDB (Time Series Database): Base de datos optimizada para almacenar series temporales (valores numéricos ligados a marcas de tiempo).
PromQL: Lenguaje de consultas usado en Prometheus para extraer y analizar métricas.
Kubernetes: Plataforma de orquestación de contenedores que automatiza la gestión, el escalado y la implementación de aplicaciones.
Alertmanager: Componente de Prometheus que gestiona alertas y notificaciones.
Pushgateway: Puerta de entrada que permite a procesos de corta duración enviar métricas a Prometheus.
Beats: Conjunto de agentes ligeros para enviar datos a Logstash o Elasticsearch.
Logstash: Herramienta de procesamiento y transformación de logs antes de enviarlos a Elasticsearch.
Filebeat: Agente diseñado para enviar archivos de log directamente a Logstash o Elasticsearch.
InfluxDB: Base de datos de series temporales usada comúnmente con Grafana para métricas de rendimiento.
Elastic: Ecosistema basado en Elasticsearch, incluye visualización (Kibana), ingestión (Beats, Logstash), y seguridad.
SIEM: Security Information and Event Management. Solución que centraliza logs, los analiza y detecta amenazas de seguridad.
